```
このリポジトリはiwakuの日常（情報）をまとめ用。

iwaku下にジャンル分けディレクトリを作成。

一つのファイルでは、足りなくなった場合はジャンルとする。
（基準は、テキスト情報を複数ファイルに分けたいかどうか。）

エンタメ/技術/健康/趣味/学習/まめ知識/お買い得
```
